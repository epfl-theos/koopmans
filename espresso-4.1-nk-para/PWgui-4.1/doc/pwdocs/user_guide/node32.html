6.3 File space requirements

A typical pw.x run will require an amount of temporary disk space in the order of O double precision complex numbers:

O = NkMN + qNr1Nr2Nr3

where q = 2 x mixing_ndim (number of iterations used in self-consistency, default value = 8) if disk_io is set to 'high' or not specified; q = 0 if disk_io='low' or 'minimal'.
