Installation
============

Downloading
^^^^^^^^^^^
Koopmans is available to download on `github <https://github.com/epfl-theos/koopmans>`_. You can clone it directly with

.. code:: bash

    git clone --recursive git@github.com:epfl-theos/koopmans.git
